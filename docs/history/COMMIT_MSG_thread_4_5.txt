Thread 4.5: Frontier++ — 3 new specialists, fusion, fail-safe, conformal LLM, adversarial harness

Closes the honest gap left after Thread 4: lexical specialists couldn't
claim the paper-SOTA ASR numerics because their own evaluation paths
were lexical-only. Thread 4.5 ships seven additions that push Tex past
Thread 4 to the genuine frontier as of May 18, 2026.

Items shipped
─────────────

1. WIRE REAL LLM JUDGES — conformal-prediction-calibrated dispatch.
   - src/tex/specialists/conformal_escalation.py — single-score CP
     primitive with split-CP finite-sample correction.
   - src/tex/specialists/llm_dispatch.py — async, provider-agnostic,
     semaphore-bounded, fail-closed dispatcher.
   - src/tex/specialists/llm_bridge.py — bridges async dispatcher to
     PlanGuard IntentLLMCallable and MAGE JudgeCallable signatures
     under the conformal escalation gate.
   - PlanGuard wires the bridge at construction time when
     TEX_SPECIALIST_LLM_MODE is set to tiered or dual_tiered.
   - Default mode is "disabled" so existing buyers see no latency drift.

2. ARGUS STANDALONE — influence-provenance graph as its own primitive.
   - src/tex/specialists/argus_specialist.py — full IPG with V_user,
     V_obs, V_decision, V_evidence node partitions and
     {derives_from, justified_by, contradicted_by} edge kinds.
   - Counterfactual test per arxiv 2605.03378v1 §3.3: rebuild IPG
     with instruction-like obs redacted, check decision retains
     justification path.
   - Four reason codes mapped to ASI01 / ASI06 / ASI09.

3. ATTRIGUARD — causal attribution via shadow replay.
   - src/tex/specialists/attriguard_specialist.py — runtime
     causal-attribution specialist per arxiv 2603.10749 (Hu et al.,
     March 2026). Re-evaluates the agent's policy under control-
     attenuated observations; flags observations whose attenuation
     would materially change the proposed action.
   - Emits REQUIRES_HUMAN_REVIEW flag on ≥ 2 driver observations.

4. VIGIL + SIREN — verify-before-commit on tool stream.
   - src/tex/specialists/vigil_specialist.py — paper-faithful
     verify-before-commit semantics per arxiv 2601.05755v2.
   - Five SIREN-class signals: intent divergence (corroborated by
     poison signal), tool-stream poisoning, cross-observation
     contamination, cascade-style chaining, metadata claim mismatch.
   - Emits REQUIRES_HUMAN_REVIEW flag on deny verdicts.

5. CROSS-SPECIALIST FUSION LAYER.
   - src/tex/specialists/fusion.py — pair signals (MAGE × AgentArmor;
     ARGUS × AttriGuard; VIGIL × ClawGuard; PlanGuard × MAGE;
     AgentArmor × AttriGuard) + cascading-failure detection (≥ 3
     specialists firing AND ≥ 1 frontier specialist).
   - Frontier-specialist agreement gets 1.5x multiplier per Nasr-class
     adaptive-attack robustness.
   - Router consumes fused_risk in place of max_risk_score.
   - 0 firing specialists = 0 corroboration bonus (preserves
     calibration; verified test passing).

6. FIVE EYES requires_human_review FLAG.
   - src/tex/specialists/human_review.py — HumanReviewEscalation
     aggregator with 4 escalation rules:
       1. explicit specialist REQUIRES_HUMAN_REVIEW flag
       2. high-risk (≥ 0.7) + structural specialist firing
       3. defense-in-depth cascade (≥ 3 specialists firing)
       4. ASI08 (cascading_failure) tagged anywhere
   - Composed onto SpecialistBundle without modifying contract surface.

7. ADVERSARIAL FUZZ HARNESS — measured ASR.
   - src/tex/adversarial/ — FuzzRunner against FastAPI TestClient.
   - 39 curated fixtures across 6 benchmarks: AgentDojo, InjecAgent,
     MCPSafeBench, AgentLAB, SIREN, Nasr-adaptive.
   - scripts/run_adversarial.py — CLI for nightly CI.
   - Converts CLAIMS from "we cite paper SOTA" to "we measure our own
     ASR against the same benchmarks the papers do."
