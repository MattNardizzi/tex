Thread 7.1 — frontier upgrades
==============================
Self-audit identified seven gaps between Thread 7 and the May 2026
literal frontier. Thread 7.1 closes all seven:

  1. Shapley-value attribution (was: heuristic). Halpern-Kleiman-Weiner
     2018 / Friedenberg-Halpern 2019 / Jorgensen et al. arxiv 2605.00248
     (Apr 30 2026). Exact for n <= 6 upstreams via the 2^n closed form;
     Castro-Gomez-Tejada 2009 unbiased Monte-Carlo with adaptive sample
     budget (120/60/30 by n band) above. Bitmask-cached payoff. p99 at
     n=20 < 5ms. FastAttribution now exposes per-candidate
     shapley_scores; top_candidates sorted by descending share.

  2. Three-dimension drift taxonomy (was: distributional only).
     Rath 2026 (arxiv 2601.04170 sec 3). DriftEvaluation now exposes
     semantic_drift / coordination_drift / behavioral_drift /
     dominant_dimension. _SIGNAL_TO_DIMENSION map routes each
     signal to one Rath axis. drift_delta = max(three).

  3. ProbGuard PCTL systemic risk scorer (was: NotImplementedError
     flag-gated). arxiv 2508.00500 v3 (Mar 27 2026). New
     tex.systemic.probguard module: 27-state DTMC abstraction over
     (agent_count, capability_pressure, compromise) bands. Online-
     learning transition matrix with Laplace add-alpha (default 0.05)
     + self-loop prior (default 50.0). Reachability via
     P[F<=k unsafe | current_state] using stdlib matrix algebra.
     Cold-start safe-state risk ~0.08; absorbing unsafe states
     return 1.0. p99 latency 0.48 ms.

  4. RiskGate Viability Index (was: not surfaced). arxiv 2604.24686
     (Apr 27 2026, Marin-Chaudhary). EcosystemAxisScores.viability_index
     computed property: 1 - max(U, SB, RG) where U=drift_delta,
     SB=max(contract_severity, 1-governance_legality), RG=systemic_risk.

  5. RiskGate P3 monotonic restriction (was: not implemented).
     Opt-in via EcosystemEngine(monotonic_restriction=True). Per-actor
     viability floor recorded across evaluations; never relaxes
     without explicit record_recovery(actor) call. PERMIT rationale
     notes when floor is enforced. Telemetry:
     ecosystem.engine.monotonic_restriction.floor_enforced.

  6. GAAT OpenTelemetry span schema (was: not framed as OTel-compat).
     Apple, arxiv 2604.05119 (Apr 6 2026). New
     tex.observability.governance_span module: verdict_to_otel_attributes
     emits the GAAT GTS sec III.A core set (governance.decision,
     governance.enforcement_level, governance.viability_index) plus
     the six Tex-specific axes. GraduatedEnforcementLevel enum
     (L0..L4) mirrors GAAT's Theorem 3 table. No hard OpenTelemetry
     SDK dependency.

  7. ProbeMapPolicy declarative classifier (was: hardcoded dict).
     Frozen dataclass with exact_rules + substring_rules tiers.
     DEFAULT_PROBE_MAP_POLICY covers ontology event kinds; operators
     register custom event kinds without modifying Tex source.
     Mirrors GAAT's OPA Rego rule layering.
