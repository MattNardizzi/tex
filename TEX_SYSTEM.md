## 2. What the system is (from the entrypoint + routes)

Deployed as `uvicorn tex.main:app` (Helm `deploy/helm/tex/templates/pdp.yaml:15`). `tex.main.build_app()` is a FastAPI application factory: it constructs a large runtime object graph (stores, the Policy Decision Point, evidence recorder + signer, discovery service + connectors, contract enforcer, ecosystem engine, vigil engine) and mounts ~24 routers exposing **121 live HTTP routes**. There is **no `pyproject.toml`/`setup.py`** in the repo (finding): the package is run from the `src/` layout directly by uvicorn, not pip-installed as declared anywhere.

In one line, from the wired code: Tex is an **agent-governance decision service** that (1) discovers/inventories agents via connectors, (2) evaluates each proposed action through a Policy Decision Point that issues **PERMIT / ABSTAIN / FORBID**, (3) records every decision into a **hash-chained, signed evidence ledger**, (4) monitors drift and systemic risk, (5) proposes calibrated policy changes (learning), and (6) speaks a single surprise-selected sentence through the **Vigil** voice surface. The headline post-quantum-crypto and zero-knowledge capabilities exist as real code but are **parked or runtime-gated by default** (see ledgers below).

## 9. Claim-vs-Code ledger (the list of overclaims; code wins)

1. Post-quantum crypto is the headline, but the **wired default signer is ECDSA-P256** (classical). `tex.evidence.seal` dispatches through algorithm-agility preferring composite-ML-DSA, but falls back to `ecdsa-p256` when no PQ backend is installed, and `tex.events._ecdsa_provider.default_signature_provider()` is hard-coded to ECDSA. Whether evidence is post-quantum-signed is **RUNTIME-DEPENDENT** on pyca≥48 (OpenSSL 3.5+) or liboqs being present. `src/tex/evidence/seal.py`, `src/tex/events/_ecdsa_provider.py:109`.

2. The ML-DSA / ML-KEM / HQC / LMS modules' **names** imply native PQ implementations; the **code** is a dispatch shim to external backends (`cryptography.hazmat...mldsa` or `oqs`). They are real wiring, not pure-Python algorithms, and most are PARKED. `src/tex/pqcrypto/ml_dsa.py:95-243`.
